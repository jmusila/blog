# Laravel Blog

## How to test the app
- clone the application `$ git clone https://github.com/jonathanmusila/blog.git`
- cd into the blog folder `$ cd blog`
- run the composer to install dependancies `$ composer install`
- run npm or yarn `$ nmp install or yarn`
